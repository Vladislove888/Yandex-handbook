# Yandex-handbook
Хэндук Яндекса "Прикладной анализ данных в социальных науках" для практики в анализе данных

Учебник составлен так, чтобы любой человек без навыков в области анализа данных или программирования мог за короткий срок понять общую логику и техническую сторону процесса, провести самостоятельное исследование и научиться программировать на Python.

**Темы, которые содержит учебник**

- Дескриптивный анализ данных. Введение в понятие данных и описательные статистики числовых данных
- Парные сравнения числовых данных. Вводятся понятия выборки и генеральной совкупности, а также описываются базовые статистические тесты
- Категориальные данные и их анализ
- Визуализация данных
- Метод главных компонент и факторный анализ
- Сбор (парсинг) и очистка данных
- Подготовка данных: хранение, форматы и кодировка данных. Обработка пропущенных значений. Приведение данных к целевому виду
- Планирование исследования, определение и тестирование гипотез
- Линейная регрессия, метод наименьших квадратов
- Логистическая регрессия
- Текстовые данные и способы их обработки

Каждая глава содержит практические задачи на Python, которые помогут закрепить материал. Однако на момент прохождения мною этого курса в этих заданиях были недочёты (например, формулировка следующего упражнения повторяет предыдущее, а ответ требуется иной), потому выложил только те, где прошёл полностью.
